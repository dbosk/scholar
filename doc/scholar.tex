\begin{abstract}
Scholar is a command-line tool for conducting structured literature searches
across multiple academic databases. It provides a unified interface for
searching Semantic Scholar, OpenAlex, DBLP, Web of Science, and IEEE Xplore,
with caching to manage API rate limits and an interactive TUI for reviewing
results.

This document is the complete literate source of Scholar, combining
documentation and implementation in a single narrative.
\end{abstract}
